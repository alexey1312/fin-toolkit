Protocol-first financial analysis toolkit with MCP server for Claude Code.

20 tools, 6 AI agents, 7 data providers, 8 search engines — works out of the box, no API keys required.

**[Documentation](https://alexey1312.github.io/fin-toolkit/)** · **[Getting Started](https://alexey1312.github.io/fin-toolkit/getting-started/)**

## Quick Start

```bash
uv tool install "fin-toolkit @ git+https://github.com/alexey1312/fin-toolkit.git"
fin-toolkit quickstart
```

## Development

```bash
git clone https://github.com/alexey1312/fin-toolkit.git && cd fin-toolkit
uv sync
fin-toolkit setup
```
